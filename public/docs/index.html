Introduction

This documentation aims to provide all the information you need to work with our Delivercare API's.

Endpoints

Generate Token

This API will allow a partner to create API token with some x hours expiry and this token will be used for all subsequent API calls to fetch the data.

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/v1/generate-token" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"username\": \"anshul\",
    \"password\": 12345678
}"

const url = new URL(
    "http://127.0.0.1:8000/api/v1/generate-token"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "username": "anshul",
    "password": 12345678
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
